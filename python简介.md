# Python简介

- Python是时下最流行、最火爆的编程语言之一，具体原因如下：

1. 简单、易学，适应人群广泛

2. 免费、开源

3. 应用领域广泛：
- 自动化测试、自动化运维、web开发、网络爬出、数据分析、人工智能、机器学习

*一下知名框架均是python语言开发 * 
- Google开源机器学习框架：TensorFlow
- 开源社区主推学习框架：Scikit-learn
- 百度开源深度学习框架：paddle
